# NoName (Unity)

- Unity: `6000.3.1f1`
- UI: `UI Toolkit`
- Input: `Input System`

## 빠른 실행

1. Unity에서 프로젝트를 열고 Play
2. 치트(에디터/개발 빌드):
   - `1` / `2`: 카드 추가
   - `N`: 랜덤 카드 추가
   - `P`: 페이즈 진행

## 샘플 정적 데이터 생성

- 메뉴: `NoName > Static Data > Create Sample Assets`
  - `Assets/NoName/Resources/NoName/StaticData/StaticDataSettings.asset` 생성
  - 이후 런타임에서 정적 데이터가 해당 에셋을 사용합니다.
